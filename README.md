# BuildWise AI - Enterprise Construction Intelligence Platform

BuildWise AI is an enterprise-grade, AI-powered Construction Intelligence Platform. It streamlines construction management by offering predictive scheduling risks, machine-learned cost estimators, automated blueprint image parsers, site safety compliance photo scanners, document intelligence OCR indexing, and a database-connected conversational AI assistant.

---

## Technical Stack

* **Frontend**: React + TypeScript + Tailwind CSS + Recharts + Vite
* **Backend**: FastAPI + SQLAlchemy + PostgreSQL + Alembic + Uvicorn
* **AI & Vision**: OpenCV + Scikit-Learn + NumPy + Pandas
* **Proxy Server**: Nginx (Reverse HTTP & WebSocket Dev proxy routing)
* **Packaging & Deploy**: Docker & Docker Compose

## Step-by-Step Setup & Deploy

Ensure you have [Docker and Docker Compose](https://www.docker.com/) installed on your machine.

### 1. Spin Up the Services
From the root directory containing `docker-compose.yml`, run the build and launch command:
```bash
docker-compose up --build
```
This builds and launches four key services in Docker:
1. **PostgreSQL Database** (`db:5432`)
2. **FastAPI Backend Web Gateway** (`backend:8000`)
3. **React + Vite Frontend Dev Portal** (`frontend:5173`)
4. **Nginx Reverse Proxy Router** (`nginx:80`)

### 2. Seed the PostgreSQL Database
Once the database container is active (and has completed health checks), open a new terminal window and run the DB Seeding script inside the backend container to pre-populate projects, workers, machinery, and budgets:
```bash
docker-compose exec backend python seed.py
```

### 3. Open the Web Application
Open your browser and navigate to:
```
http://localhost
```
You will be greeted by the BuildWise AI login portal.

## Key Features Guide

### 1. Unified Dashboard
- Visualizes allocated budget vs actual spending across material, labor, equipment, and overhead categories using Recharts.
- Monitors live site weather telemetry and shows concrete-pouring feasibility advice.
- Summarizes composition risk scores generated by the predictive AI classifier.

### 2. Project Board & Gantt Roadmaps
- Houses three sub-views: Kanban Task Board, Milestones Roadmap, and horizontal Gantt Schedules.
- Create new tasks and assign them to users, managing dependencies and due dates.

### 3. Materials & Heavy Machinery Logs
- Highlights materials running low on-site (quantities below minimum required threshold).
- Simulate purchase orders using the **Purchase Request** form.
- Toggle heavy machinery (e.g. Cranes, Excavators) states between "Available" and "Maintenance" to trigger risk changes.

### 4. Machine Learning cost & Risk engines
- **Cost Estimation**: Estimates material, labor, and permit allocations using a `RandomForestRegressor` model trained on square footage, finish class, and labor indices.
- **Risk Prediction**: Analyzes database metrics (overdue task ratios, budget variances, material deficits, and active machine maintenance ratios) and feeds parameters to a `RandomForestClassifier` to estimate composite delay factors.

### 5. Computer Vision & Blueprint Analytics
- Upload schematic floor plans (PDF or PNG) to detect doors, windows, and partition sizes using OpenCV edges and line contour transforms. Click Side cards to highlight rooms on the canvas.
- Upload site photos to detect helmet/vest compliance rates and concrete crack or water leakage hazards.

### 6. Document Intelligence & AI Chat
- Upload invoice or contract drawings to run OCR indexers, extracting dates, amounts, vendors, and agreement clauses.
- Converse with the AI Assistant using natural language query suggestions ("Why is my project delayed?") to analyze active database records.
